# Diabetes 30-Day Hospital Readmission Analysis

**Predicting early readmission risk for diabetic patients using clinical and administrative data**

## Project Overview

Analysis of 101,766 hospital encounters from 130 US hospitals (1999-2008) to identify factors driving 30-day readmissions in diabetic patients. This project combines my clinical expertise as a trauma step-down RN with data science techniques to address a critical healthcare quality metric.

## Dataset

- **Source:** [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/296/diabetes+130-us+hospitals+for+years+1999-2008)
- **Size:** 101,766 encounters, 71,518 unique patients
- **Features:** Demographics, admission details, lab procedures, medications, diagnosis codes
- **Target:** 30-day hospital readmission (11% positive class)

## Key Findings

- **Prior inpatient visits** are the strongest predictor of readmission (r=0.165)
- **A1C testing** (quality indicator) correlates with lower readmission rates
- Demographics (age, gender, race) show no significant differences
- Readmission is driven by patient history and care quality, not current hospitalization metrics
